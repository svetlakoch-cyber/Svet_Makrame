# Сайт‑выставка мастера макраме

Простой одностраничный сайт‑выставка мастера макраме Светланы Кочарян, сверстанный на чистом HTML и CSS.

## Структура

- `index.html` — разметка страницы (блоки: «Обо мне», услуги, портфолио, отзывы, контактная форма).
- `index.html` — разметка страницы (блоки: «Обо мне», «Изделия», «Галерея», «Отзывы», форма заказа).
- `styles.css` — стили и адаптивная верстка.

## Как запустить

1. Откройте файл `index.html` двойным кликом или через контекстное меню «Открыть с помощью…» → браузер.
2. Для дальнейшей доработки редактируйте тексты прямо в `index.html`, стили — в `styles.css`.

## Как адаптировать под себя

- Замените тексты блоков (имя, ниши, цифры, описания проектов и отзывов) в `index.html`.
- При необходимости поменяйте цветовую схему в `styles.css` (CSS‑переменные в блоке `:root`).
